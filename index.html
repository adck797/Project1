Bootcamp 2021: Fly box

My first programming project, concluding the 2 week Programming Bootcamp at City (2021). This Processing project demonstrates a comprehensive understanding of the use of arrays, loops, and randomness. I created a fly box in which details of each pillar (e.g. position, pillar color, bird colour and size, crown color and shape) are stored in parallel 2D arrays. when the bird passes through the pillars the point score updates and when ever the bird hits the pillar the programme will display a message saying game over, the score you have recived, and letting you know that by pressing the spacebar button you can try again. (which is a loop that i have created)
